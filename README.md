## ✨ Features

- **Scroll Animations** — Elements animate in as you scroll, powered by Intersection Observer
- **Smooth Scrolling** — Physics-based momentum scrolling via `scrollice.js`
- **Dropdown Navigation** — Hover-activated menus with clean transitions
- **Custom Right-Click Menu** — Context menu with copy-to-clipboard support
- **Offline Support** — Service worker caches assets for offline viewing
